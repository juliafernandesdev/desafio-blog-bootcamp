O que é HTML?

(Voltar)

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto, em inglês. Foi criada pelo físico britânico, Tim Beners-Lee, na década de 1990, com o objetivo de ajudar na formatação de documentos de pesquisa compartilhados entre ele e seus colegas.

A partir da popularização da internet e o desenvolvimento dos navegadores (browsers), o HTML começou a ser utilizado na criação de sites, o que o tornou um dos pilares da "World Wide Web" (www).

O que são tags em HTML?

(Voltar)

As tags possuem uma estrutura própria, iniciando com o sinal “menor que” ("<"), seguido do nome daquele elemento e finalizando com o sinal “maior que” (">"). Elas são classificadas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). Para identificar o fechamento de uma tag utilizad-se uma barra (/), porém para as tags de auto fechamento, não existe necessidade da presença desse caractere.

Inicialmente, as tags podem ser divididas em dois tipos: em “nível de bloco” (block-level) e “em linha” (inline). Quando tem-se um elemento em nível de bloco, ele ocupa toda a largura de seu elemento pai, o qual também chamamos de elemento container, criando assim um “bloco”. Já quando tem-se elementos inline, normalmente utilizamos para demarcação de conteúdos de texto.

Tags que aprendemos até agora

(Voltar)

HTML -> representa a raiz do documento, onde engloba todos os outros elementos HTML.
body -> é o local onde fica todo o conteúdo de textos, imagens, vídeos, e o usuário interage. Nele, os conteúdos são estruturados pelas demais tags do HTML.
head -> engloba as informações do documento que serão interpretadas pelo navegador.
h1, h2, h3, h4, h5, h6 -> tags de tipografia do HTML. Utilizadas para inserir títulos no conteúdo.
p -> insere um parágrafo no conteúdo.
br -> insere uma quebra de linha.
strong -> coloca o conteúdo demarcado em negrito.
em -> coloca o conteúdo demarcado em itálico.
ol -> cria uma lista ordenada.
ul -> cria uma lista não ordenada.
li -> tag filha da ol/ul, onde identifica os itens da lista.
